Bridges the Anthropic and OpenAI APIs, enabling Claude Code to work with any OpenAI-compatible model. Runs on Tanzu Platform.

How to connect Claude Code:
  1. Click Copy on any model below
  2. Paste the commands into your terminal
  3. Claude Code will connect through the bridge to the selected model
Requires claude CLI installed. Install at claude.ai/claude-code.
Model context window requirements
Claude Code sends a large system prompt (~7K tokens) plus 25 tool definitions (~20K tokens). The model must have enough context to fit this overhead plus your conversation.
ContextSuitability
32K tokensMinimum — short conversations only
64K tokensRecommended — comfortable for coding sessions
128K+ tokensIdeal — extended sessions without hitting limits
If a model returns 500 errors, ask your platform operator to increase the context window limit in Tanzu Ops Manager.
Managing context with custom models
Claude Code doesn't know the context window of custom models — it only tracks built-in Claude models. This means it won't automatically compact the conversation when your model's limit is approaching. You need to manage context manually:
CommandWhen to use
/compactSummarizes the conversation to free up context. Use when responses slow down or the model starts returning errors mid-session. You can add a focus area: /compact focus on the API changes
/clearResets the conversation entirely. Use when switching tasks or starting fresh work.
/contextShows what's using space in your context window. Useful to check before compacting.
